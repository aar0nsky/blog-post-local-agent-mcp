# Documentation Index

Welcome — this repository contains a set of guides and examples to help you run a local LLM-based assistant using Ollama, Continue.dev and Model Context Protocol (MCP) servers.

Clone the repo to follow along:

```bash
git clone https://github.com/aar0nsky/blog-post-local-agent-mcp.git
cd blog-post-local-agent-mcp
```

Quick links (absolute URLs):

- **Getting started — Environment Setup**: https://github.com/aar0nsky/blog-post-local-agent-mcp/blob/main/docs/environment-setup.md
  - A step‑by‑step guide to installing prerequisites (Git, Docker, Node.js, Python, VS Code), setting up Ollama and Continue, and preparing your machine for the examples in this repo.

- **MCP Servers Guide**: https://github.com/aar0nsky/blog-post-local-agent-mcp/blob/main/docs/mcp-servers-guide.md
  - Overview of the available MCP servers in this project (Fetch, Filesystem, Git, SQLite, XRAY, etc.), recommended configurations and example prompts.

- **Maintenance & Upgrades**: https://github.com/aar0nsky/blog-post-local-agent-mcp/blob/main/docs/maintenance-and-upgrades.md
  - How to keep Docker images, Ollama models and Continue up to date, plus tips for adding or upgrading MCP servers safely.

- **PoC Demo Walkthrough**: https://github.com/aar0nsky/blog-post-local-agent-mcp/blob/main/docs/poc-demo.md
  - A guided demo using the `examples/poc-local-ai-workflow` project to try out refactors, PR triage, and agent workflows locally.

- **Repository README**: https://github.com/aar0nsky/blog-post-local-agent-mcp/blob/main/README.md
  - High level overview of the project and links to examples and scripts.

- **Examples — PoC Local AI Workflow**: https://github.com/aar0nsky/blog-post-local-agent-mcp/blob/main/examples/poc-local-ai-workflow/README.md
  - Example app, utilities and instructions used by the PoC demo referenced throughout the docs.
